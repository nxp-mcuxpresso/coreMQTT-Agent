MCUXpresso SDK: coreMQTT Agent Library NXP Integration
======================================================

This repository is the NXP integration layer for the coreMQTT Agent library (https://github.com/FreeRTOS/coremqtt-agent). It allows original coreMQTT Agent upstream sources build in MCUXpresso SDK. It is part of the MCUXpresso SDK overall delivery which is composed of several sub-repositories/projects. Navigate to the top/parent repository mcuxsdk-manifests(https://github.com/nxp-mcuxpresso/mcuxsdk-manifests) for the complete delivery of MCUXpresso SDK.
